# 個人作品集 Portfolio

本專案為個人作品集網站，採用 **純 HTML + CSS（無建置工具）** 開發，
設計風格偏向 AI Engineer / Full Stack Developer 專業定位，
可直接部署於 **GitHub Pages**，無需額外打包或編譯流程。

---

## 🔗 線上預覽（GitHub Pages）

本網站已部署於 GitHub Pages，可透過以下網址直接瀏覽：

```text
https://manipulateed.github.io/
```

---

## 🚀 部署方式

此專案為靜態網站，只需將 `index.html` 推送至 `main` 分支，
並在 Repository 設定中啟用 **GitHub Pages（Deploy from main branch）** 即可完成部署。
